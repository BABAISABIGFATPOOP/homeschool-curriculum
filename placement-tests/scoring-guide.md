# Placement Test Scoring Guide

For each subject, find where the student starts struggling consistently:

| Comfortable through... | Start at... |
|---|---|
| Section A only | **Grade 4** |
| Section A + some of B | **Grade 5** |
| Most of Section B | **Grade 6** |
| All of B + some of C | **Grade 7** |
| Most of Section C | **Grade 8** |
| All of C + some of D | **Grade 9** |
| Most of Section D | **Grade 10** |

## Notes

- It's completely normal to place at different levels for different subjects. For example, a student might be Grade 7 in Math but Grade 5 in English — just start each subject where they placed.
- If a student is borderline between two levels, start at the lower one. It's better to build confidence with review than to struggle with gaps.
- These tests are meant as a rough guide, not a precise measurement. Adjust as needed once you start the curriculum.
