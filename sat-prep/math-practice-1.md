### Answer Explanations for Selected Questions

**Question 4:** f(-1) = 2(-1)² - 5(-1) + 3 = 2(1) + 5 + 3 = 2 + 5 + 3 = 10

**Question 12:** (a/b)(b/c) = a/c = (3/5)(5/7) = 3/7

**Question 17:** Rearranging 4x - 2y = -6 gives -2y = -4x - 6, so y = 2x + 3. This is the same as the first equation, so the system has infinitely many solutions.

**Question 18:** Let width = w. Length = w + 4. Perimeter: 2(w + w + 4) = 56, so 4w + 8 = 56, 4w = 48, w = 12. Length = 16. Area = 12 × 16 = 192.

**Question 20:** Total distance = 180 + 240 = 420 miles. Time for first part = 3 hours. Time for second part = 240/80 = 3 hours. Total time = 6 hours. Average speed = 420/6 = 70 mph.

**Question 32:** h(t) = -16t² + 64t + 80. The vertex is at t = -64/(2(-16)) = -64/(-32) = 2. h(2) = -16(4) + 64(2) + 80 = -64 + 128 + 80 = 144 feet.

**Question 38:** Discounted price: $120 × 0.85 = $102. With tax: $102 × 1.08 = $110.16.

**Question 46:** a/4 = b/7, so 7a = 4b, and b = 7a/4. Substituting: a + 7a/4 = 33, so 11a/4 = 33, and a = 12.

**Question 54:** sin²θ + cos²θ = 1. (5/13)² + cos²θ = 1. 25/169 + cos²θ = 1. cos²θ = 144/169. cos θ = 12/13 (positive in first quadrant).

**Question 57:** y = 2(0 - 3)² - 8 = 2(9) - 8 = 18 - 8 = 10. The y-intercept is (0, 10).

**Question 58:** The vertex of P(x) = -2x² + 40x - 150 is at x = -40/(2(-2)) = -40/(-4) = 10.
