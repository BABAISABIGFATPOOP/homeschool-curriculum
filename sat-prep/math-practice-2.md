### Answer Explanations for Selected Questions

**Question 3:** Testing n = 10: C = 7.50(10) + 20 = 95. Testing n = 20: C = 7.50(20) + 20 = 170. Both match the table.

**Question 4:** f(3) = -(3)² + 6(3) - 5 = -9 + 18 - 5 = 4

**Question 11:** From x - y = 1, x = y + 1. Substituting: 2(y + 1) + 3y = 18 → 2y + 2 + 3y = 18 → 5y = 16 → y = 3.2

**Question 15:** Original mixture: 12 liters, of which 40% is OJ = 4.8 liters OJ. After adding 3 liters OJ: 7.8 liters OJ in 15 liters total. 7.8/15 = 0.52 = 52%.

**Question 19:** (a + b)² = a² + 2ab + b² = 49. Since ab = 10, we have a² + b² + 20 = 49, so a² + b² = 29.

**Question 22:** Markup: ×1.60. Discount: ×0.75. Combined: 1.60 × 0.75 = 1.20. That's a 20% increase.

**Question 31:** f(2) = 2(2) + 1 = 5. g(5) = 5² - 3 = 25 - 3 = 22.

**Question 43:** For exactly one real solution, discriminant = 0. (-12)² - 4(k)(9) = 0 → 144 - 36k = 0 → k = 4.

**Question 44:** d = √((5-(-3))² + (-2-4)²) = √(64 + 36) = √100 = 10.

**Question 46:** Space diagonal = √(8² + 6² + 4²) = √(64 + 36 + 16) = √116.

**Question 52:** Sum of 8 numbers = 8 × 15 = 120. Remove 23: sum = 97. Average = 97/7 ≈ 13.86.

**Question 57:** f(x) = a(x + 3)(x - 7). At x = 0: a(3)(-7) = -42 → -21a = -42 → a = 2. So f(x) = 2(x + 3)(x - 7) = 2x² - 8x - 42.

**Question 58:** y = 3.2(8) + 45 = 25.6 + 45 = 70.6 (in thousands) = $70,600.
