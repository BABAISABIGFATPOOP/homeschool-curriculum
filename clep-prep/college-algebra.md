# College Algebra CLEP Study Guide

## Exam Overview

| Detail | Info |
|---|---|
| **Format** | Multiple choice |
| **Number of Questions** | 60 |
| **Time Limit** | 90 minutes |
| **Passing Score** | ~50 out of 80 (scaled); approximately 50% correct |
| **College Credit** | Typically 3 semester hours |
| **Calculator** | On-screen graphing calculator provided |

This exam tests the material from a one-semester college algebra course. It goes deeper into algebraic topics than the College Mathematics CLEP but is narrower in scope.

---

## Topics Covered

### 1. Equations and Inequalities (25%)

- **Linear equations**: solving, word problems, literal equations
- **Quadratic equations**: factoring, completing the square, quadratic formula
- **Absolute value equations and inequalities**: |ax + b| = c, |ax + b| < c
- **Polynomial equations**: solving by factoring, rational root theorem
- **Rational equations**: finding LCD, checking for extraneous solutions
- **Radical equations**: isolating the radical, squaring both sides, checking solutions
- **Linear inequalities**: solving, graphing on a number line, interval notation
- **Quadratic and polynomial inequalities**: sign charts, test intervals
- **Systems of linear inequalities**

**Key Concepts:**
- Quadratic formula: x = [-b ± sqrt(b² - 4ac)] / (2a)
- Discriminant (b² - 4ac): positive = 2 real solutions, zero = 1 real solution, negative = no real solutions
- When multiplying/dividing an inequality by a negative number, flip the sign
- Always check solutions in rational and radical equations for extraneous results

### 2. Functions (35%)

#### Linear Functions
- Slope-intercept form: y = mx + b
- Point-slope form: y - y₁ = m(x - x₁)
- Standard form: Ax + By = C
- Parallel lines have equal slopes; perpendicular lines have negative reciprocal slopes
- Direct variation: y = kx

#### Quadratic Functions
- Standard form: f(x) = ax² + bx + c
- Vertex form: f(x) = a(x - h)² + k
- Vertex: (-b/(2a), f(-b/(2a)))
- Axis of symmetry: x = -b/(2a)
- Opens up if a > 0, down if a < 0
- Maximum/minimum value is the y-coordinate of the vertex

#### Polynomial Functions
- Degree determines end behavior and maximum number of turning points
- Leading coefficient test for end behavior
- Zeros/roots and their multiplicities
- Intermediate Value Theorem
- Factor theorem: (x - c) is a factor if f(c) = 0
- Remainder theorem: f(c) = remainder when dividing by (x - c)
- Synthetic division and long division

#### Rational Functions
- f(x) = p(x)/q(x) where q(x) ≠ 0
- Domain: exclude values where denominator = 0
- Vertical asymptotes: where denominator = 0 (after simplification)
- Horizontal asymptotes: compare degrees of numerator and denominator
  - Degree of numerator < denominator: y = 0
  - Degrees equal: y = leading coefficient ratio
  - Degree of numerator > denominator: no horizontal asymptote (may have slant)
- Holes: common factors in numerator and denominator

#### Exponential Functions
- f(x) = a * b^x where b > 0, b ≠ 1
- Growth if b > 1, decay if 0 < b < 1
- Natural exponential: f(x) = e^x
- Compound interest: A = P(1 + r/n)^(nt)
- Continuous growth/decay: A = Pe^(rt)
- Horizontal asymptote at y = 0 (for basic form)
- Always passes through (0, a)

#### Logarithmic Functions
- y = log_b(x) means b^y = x
- Common log: log(x) = log₁₀(x)
- Natural log: ln(x) = log_e(x)
- Properties:
  - log_b(MN) = log_b(M) + log_b(N)
  - log_b(M/N) = log_b(M) - log_b(N)
  - log_b(M^p) = p * log_b(M)
  - log_b(1) = 0, log_b(b) = 1
- Change of base: log_b(x) = ln(x) / ln(b)
- Domain: x > 0

**Key Concepts for All Functions:**
- Domain and range identification
- Function notation and evaluation
- Composition: (f ∘ g)(x) = f(g(x))
- Inverse functions: switch x and y, solve for y
- Transformations: vertical/horizontal shifts, reflections, stretches/compressions
- Even functions: f(-x) = f(x), symmetric about y-axis
- Odd functions: f(-x) = -f(x), symmetric about origin

### 3. Systems of Equations (15%)

- **Two-variable linear systems**: substitution, elimination, graphing
- **Three-variable linear systems**: elimination, back-substitution
- **Non-linear systems**: substitution method (one linear, one quadratic)
- **Matrices**: augmented matrices, row echelon form, Gaussian elimination
- **Determinants**: 2x2 and 3x3
- **Cramer's Rule**
- **Types of solutions**: one solution (consistent/independent), infinitely many (dependent), no solution (inconsistent)

**Key Concepts:**
- 2x2 determinant: |a b; c d| = ad - bc
- Cramer's Rule: x = D_x/D, y = D_y/D (where D ≠ 0)
- Parallel lines = no solution; same line = infinitely many

### 4. Graphing and Analysis (25%)

- **Coordinate plane basics**: plotting points, quadrants, distance, midpoint
- **Graphing techniques**: intercepts, symmetry, transformations
- **Circles**: (x - h)² + (y - k)² = r², center (h, k), radius r
- **Parabolas, ellipses, hyperbolas** (basic recognition and graphing)
- **Piecewise functions**: graphing and evaluating
- **Absolute value functions**: V-shaped graph, vertex
- **Increasing, decreasing, and constant intervals**
- **Relative maxima and minima**
- **End behavior of polynomial and rational functions**

**Key Concepts:**
- Distance formula: d = sqrt[(x₂-x₁)² + (y₂-y₁)²]
- Midpoint: ((x₁+x₂)/2, (y₁+y₂)/2)
- Transformation order: horizontal shift, stretch/compress, reflect, vertical shift
- To graph f(x - 2) + 3: shift right 2, up 3

---

## Study Tips

1. Functions make up the largest portion (35%) — master all six types.
2. Practice converting between forms (e.g., standard to vertex for quadratics).
3. Know logarithm properties cold — they appear frequently.
4. Understand transformations thoroughly; they apply to every function type.
5. Practice solving systems using multiple methods.
6. Use the graphing calculator strategically but understand the math behind it.

## Practice Questions

**1.** Solve: 3x² - 12x + 9 = 0

A) x = 1, x = 3
B) x = -1, x = -3
C) x = 1, x = -3
D) x = 3 only

**2.** What is the vertex of f(x) = 2(x - 3)² + 5?

A) (3, 5)
B) (-3, 5)
C) (3, -5)
D) (-3, -5)

**3.** Simplify: log₂(8) + log₂(4)

A) 5
B) 12
C) 7
D) 32

**4.** What is the horizontal asymptote of f(x) = (3x² + 1) / (x² - 4)?

A) y = 0
B) y = 3
C) y = -4
D) No horizontal asymptote

**5.** If f(x) = 2x + 1 and g(x) = x² - 3, what is (g ∘ f)(2)?

A) 7
B) 22
C) 5
D) 3

**6.** Solve the system: 2x + y = 7 and x - y = 2

A) (3, 1)
B) (2, 3)
C) (1, 5)
D) (4, -1)

**7.** What is the domain of f(x) = sqrt(3x - 6)?

A) x ≥ 2
B) x > 2
C) x ≥ 6
D) All real numbers

**8.** Solve: |2x - 5| = 9

A) x = 7 or x = -2
B) x = 7 or x = 2
C) x = 7 only
D) x = -2 only

**9.** What is the inverse of f(x) = (x - 4) / 3?

A) f⁻¹(x) = 3x + 4
B) f⁻¹(x) = 3/(x - 4)
C) f⁻¹(x) = (x + 4)/3
D) f⁻¹(x) = 3x - 4

**10.** Which describes the end behavior of f(x) = -2x³ + x?

A) As x → ∞, f(x) → ∞; as x → -∞, f(x) → -∞
B) As x → ∞, f(x) → -∞; as x → -∞, f(x) → ∞
C) As x → ∞, f(x) → ∞; as x → -∞, f(x) → ∞
D) As x → ∞, f(x) → -∞; as x → -∞, f(x) → -∞

---

## Answer Key

1. **A) x = 1, x = 3** — Factor: 3(x² - 4x + 3) = 3(x - 1)(x - 3) = 0.
2. **A) (3, 5)** — In vertex form f(x) = a(x - h)² + k, the vertex is (h, k) = (3, 5).
3. **A) 5** — log₂(8) = 3 and log₂(4) = 2, so 3 + 2 = 5.
4. **B) y = 3** — Degrees are equal, so HA = ratio of leading coefficients: 3/1 = 3.
5. **B) 22** — f(2) = 5, then g(5) = 25 - 3 = 22.
6. **A) (3, 1)** — Add equations: 3x = 9, x = 3, then y = 1.
7. **A) x ≥ 2** — The expression under the radical must be ≥ 0: 3x - 6 ≥ 0, x ≥ 2.
8. **A) x = 7 or x = -2** — Case 1: 2x - 5 = 9, x = 7. Case 2: 2x - 5 = -9, x = -2.
9. **A) f⁻¹(x) = 3x + 4** — Set y = (x-4)/3, swap x and y: x = (y-4)/3, solve: y = 3x + 4.
10. **B) As x → ∞, f(x) → -∞; as x → -∞, f(x) → ∞** — Odd degree with negative leading coefficient: falls right, rises left.
