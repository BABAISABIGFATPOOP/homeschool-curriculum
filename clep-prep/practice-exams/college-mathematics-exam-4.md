# CLEP College Mathematics — Practice Exam 4

**Time Limit:** 90 minutes
**Number of Questions:** 60
**Format:** Multiple choice (A, B, C, D)
**Passing Score:** Typically 50 (on CLEP's 20–80 scale); approximately 50% correct is a reasonable target, but check with your institution.
**Calculator:** A scientific (nongraphing) calculator is provided on-screen during the real exam.

---

## Questions

**1.** If A = {2, 4, 6, 8} and B = {1, 2, 3, 4}, what is A ∪ B?

A) {2, 4}
B) {1, 2, 3, 4, 6, 8}
C) {6, 8}
D) {1, 3}

**2.** What is the contrapositive of "If a number is even, then it is divisible by 2"?

A) If a number is divisible by 2, then it is even.
B) If a number is not divisible by 2, then it is not even.
C) If a number is not even, then it is not divisible by 2.
D) A number is even and divisible by 2.
