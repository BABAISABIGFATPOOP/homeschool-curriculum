# CLEP College Algebra — Full Practice Exam

**Time Limit:** 90 minutes
**Number of Questions:** 60
**Format:** Multiple choice (A, B, C, D)

### Answer Explanations (Selected)

- **3.** |2x−5|=9 → 2x−5=9 gives x=7; 2x−5=−9 gives x=−2.
- **9.** Quadratic formula: x = (3 ± √(9+40))/4 = (3 ± 7)/4, giving x = 5/2 or x = −1.
- **10.** x = −(−6)/(2·1) = 3; y = 9−18+5 = −4. Vertex: (3, −4).
- **18.** By the Remainder Theorem, f(1) = 1+2−5+1 = −1.
- **25.** Squaring: 2x−1 = x²−4x+4 → x²−6x+5=0 → (x−1)(x−5)=0. Check: x=1 gives √1=−1 (false). x=5 gives √9=3 (true).
- **37.** log(x(x−3))=1 → x²−3x=10 → x²−3x−10=0 → (x−5)(x+2)=0. x=−2 invalid (log of negative).
- **50.** (x+2)/(x−3) = 5 → x+2=5x−15 → 17=4x → x=17/5 (valid since ≠3).
- **58.** 4^(x−1)=8 → 2^(2(x−1))=2³ → 2x−2=3 → x=5/2.
- **60.** g(f(x)) = √(x²+1−1) = √(x²) = |x|. Domain is all real numbers.
