# College Mathematics CLEP Practice Exam 3

**Time Limit:** 90 minutes
**Number of Questions:** 60
**Format:** Multiple choice (A, B, C, D)
**Passing Score:** Typically 50 (on CLEP's 20–80 scale); approximately 50% correct is a reasonable target, but check with your institution.
**Calculator:** A scientific (nongraphing) calculator is provided on-screen during the real exam.

---

## Questions

**1.** If set X = {a, b, c, d} and set Y = {c, d, e, f}, what is X ∩ Y?

A) {a, b, e, f}
B) {c, d}
C) {a, b, c, d, e, f}
D) {c}

**2.** If the universal set U = {1, 2, 3, ..., 10} and B = {2, 5, 8}, what is B'?

A) {1, 3, 4, 6, 7, 9, 10}
B) {2, 5, 8}
C) {1, 2, 3, 4, 5}
D) {3, 6, 9}

**3.** Which logical connective makes "p ___ q" true when at least one of p or q is true?

A) AND
B) OR
C) IF...THEN
D) IF AND ONLY IF

**4.** How many proper subsets does a set with 3 elements have?

A) 3
B) 6
C) 7
D) 8

**5.** The statement "Some birds cannot fly" is the negation of:

A) No birds can fly.
B) All birds can fly.
C) Some birds can fly.
D) No birds cannot fly.

**6.** A survey of 200 people found that 120 drink coffee, 80 drink tea, and 40 drink both. How many drink coffee or tea (or both)?

A) 240
B) 200
C) 160
D) 120

**7.** The inverse of "If p, then q" is:

A) If q, then p
B) If not p, then not q

D) Not p and q

**8.** If f(x) = −2x² + 5, what is f(3)?

D) −23

**11.** A function f is onto (surjective) if:

A) Each element of the domain maps to a unique element of the codomain
B) Every element of the codomain is mapped to by at least one element of the domain
C) The function passes the vertical line test
D) f(x) = f(−x) for all x
