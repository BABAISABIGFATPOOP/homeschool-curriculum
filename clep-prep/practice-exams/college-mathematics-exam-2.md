# College Mathematics CLEP Practice Exam 2

**Time Limit:** 90 minutes
**Number of Questions:** 60
**Format:** Multiple choice (A, B, C, D)
**Passing Score:** Typically 50 (on CLEP's 20–80 scale); approximately 50% correct is a reasonable target, but check with your institution.
**Calculator:** A scientific (nongraphing) calculator is provided on-screen during the real exam.

---

## Questions

**1.** If set A = {2, 4, 6, 8} and set B = {1, 2, 3, 4}, what is A ∪ B?

A) {2, 4}
B) {1, 2, 3, 4, 6, 8}
C) {1, 3, 6, 8}
D) {2, 4, 6}

**2.** If the universal set U = {1, 2, 3, 4, 5, 6, 7, 8} and A = {1, 3, 5, 7}, what is |A'|?

A) 4
B) 8
C) 3
D) 5

**3.** The contrapositive of "If p, then q" is:

A) If q, then p
B) If not q, then not p
C) If not p, then not q
D) p and not q

**4.** How many elements are in the power set of a set with 5 elements?

A) 5
B) 10
C) 25
D) 32

**5.** What is the truth value of the conjunction "3 > 2 AND 5 < 1"?

A) True
B) False
C) Undefined
D) Cannot be determined

**6.** In a class of 80 students, 45 play soccer, 30 play basketball, and 10 play both. How many play neither sport?

A) 5
B) 10
C) 15
D) 20

**7.** Which of the following is the converse of "If it snows, then school is cancelled"?

A) If school is cancelled, then it snows.
B) If school is not cancelled, then it does not snow.
C) If it does not snow, then school is not cancelled.
D) It snows and school is not cancelled.

**8.** If g(x) = 5x + 2, what is g(−3)?

D) −11

B) All real numbers except x = 4
