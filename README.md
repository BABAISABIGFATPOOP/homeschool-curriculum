## How to Use

1. Start with the **placement tests** to figure out what grade level to begin at for each subject.
2. Use the **scoring guide** to determine placement.
3. Follow the **curriculum** for each subject at the appropriate grade level.
4. It's completely normal to be at different grade levels for different subjects.
